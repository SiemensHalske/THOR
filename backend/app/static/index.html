THOR - Environmental Monitoring Project

Welcome to the THOR project, where we monitor environmental data using sensors and visualize it with modern web technologies.

Project Requirements

Must-Have Requirements:

  • Data Transmission: ESP32 sends environmental data from BME280 sensor to the backend server.
  • Backend Server: Python with Flask and Waitress or Gunicorn.
  • Frontend: React and Next.JS based frontend to display current sensor data.
  • Visualizations: Implementation of charts to visualize data over time.

Nice-to-Have Requirements:

  • Integration of additional sensors for more environmental data.
  • Complex analysis and visualization features like heatmaps or long-term trends.
  • Notification system for threshold values (e.g., high temperatures).
  • Optimization of data transmission for efficiency and reliability.
